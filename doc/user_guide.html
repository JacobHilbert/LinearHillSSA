Model description

We have

  • SSS chemical species on the state vector xSx_SxS​.

  • NNN reactions, each one with a reaction vector such that xS+RSx_S + R_SxS​+RS​ is the new each vector after the reaction, collected as rows in a matrix RNSR_{NS}RNS​.

    • A full linear dependence term of the propensities on the state vector: ANSxS+bNA_{NS} x_S +b_NANS​xS​+bN​

  • The Hill rate function Hill(x,v,α,n)=vxnxn+αn\text{Hill}(x,v,\alpha,n) = v \frac{x^n}{x^n + \alpha^n}Hill(x,v,α,n)=vxn+αnxn​ where nnn is called the hill factor; positive for activation, negative for repression.

    • This Hill-type interactions are expressed in matrix notation as Hill(TNSxS,vS,αS,nS)\text{Hill}(T_{NS} x_S,v_S, \alpha_S,n_S)Hill(TNS​xS​,vS​,αS​,nS​) and the operations are elementwise.

So the propensities are calculated as P(RNS)=ANSxS+bN+Hill(TNSxS,vN,αN,nN)P(R_{NS}) = A_{NS} x_S +b_N + \text{Hill}(T_{NS} x_S,v_N, \alpha_N,n_N)P(RNS​)=ANS​xS​+bN​+Hill(TNS​xS​,vN​,αN​,nN​)

(or if you don't like my matrix notation:) P(R)=Ax⃗+b⃗+Hill(Tx⃗,v⃗,α⃗,n⃗)P(R) = A \vec{x} + \vec{b} + \text{Hill}(T \vec{x},\vec{v},\vec{\alpha},\vec{n})P(R)=Ax+b+Hill(Tx,v,α,n)
